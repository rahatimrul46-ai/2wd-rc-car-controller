# 🚗 2WD RC Car Controller using Python & Arduino

This project is a GUI-based control system for a 2WD robotic car using **Python (Tkinter)** and **Arduino (PyFirmata)**.  
It allows real-time control of the car through a simple and interactive desktop interface.

---

## 🔧 Features
- 🎮 Forward, Backward, Left, Right control
- ⚡ PWM-based speed control
- 🖥️ User-friendly GUI (Tkinter)
- 🔌 Real-time communication with Arduino
- 🚀 Differential drive control (2WD system)

---

## 🛠️ Hardware Requirements
- Arduino (Uno/Nano)
- L298N Motor Driver
- 2 DC Motors (2WD chassis)
- Battery pack
- Jumper wires

---

## 💻 Software Requirements
- Python 3.x
- pyfirmata
- tkinter
